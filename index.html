Sistem pemesanan bahan ajar UT berbasis Vue.js

Aplikasi ini dibuat untuk Tugas Praktik 3. Fitur utama meliputi pengelolaan stok bahan ajar, pembuatan Delivery Order, dan tracking pengiriman DO. Struktur kode sudah memakai Vue Component, Template, data binding, directive, filter, computed property, watcher, dan event keyboard.

📦

Manajemen Stok

Tambah, edit, hapus, filter dependent options, sort data, status aman/menipis/kosong, dan tooltip catatan HTML.

🧾

Pemesanan Bahan Ajar

Nomor DO otomatis, pilihan paket, detail isi paket, total harga otomatis, dan validasi input sederhana.

🚚

Tracking DO

Pencarian dengan nomor DO atau NIM, Enter untuk submit, Esc untuk reset, serta update progress perjalanan.
